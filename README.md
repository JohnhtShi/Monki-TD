# Monki TD

A web based baed2D tower defense game built with vanilla JavaScript and WebGL shader backgrounds. 
<img width="1490" height="937" alt="image" src="https://github.com/user-attachments/assets/58f73c0a-67b2-496d-973f-a3311c2aa149" />

## Credits

Awesome background shaders from shader collection by Xor.
https://xordev.com

## Features

- WebGL animated shader backgrounds 
- buyable/upgradeable towers with unique abilities
- Dynamic enemy waves with scaling difficulty
- Pan & zoom canvas map
- In-game gold, lives, and wave HUD

## Getting Started

```bash
npm install
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173).
